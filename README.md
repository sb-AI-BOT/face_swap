### Git Clone
```sh
git clone https://github.com/sb-AI-BOT/face_swap.git
```

## Installation
### Requirements
* `pip install -r requirements.txt`

Note: See [requirements.txt](requirements.txt) for more details.

### Swap Your Face
```sh
python faceswap.py --src_img  --dst_img
```
